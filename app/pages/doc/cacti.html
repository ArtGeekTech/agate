用途

该接口用于前端把收集好的数据记录到后端服务器上，并最后收集绘制出CACTI的监控图表。

接口地址：

http://m.ued.qunar.com/monitor/log

参数：

参数 类型 必填 备注
code String 是 监控名，参考规范
time int -- 时间（监控图的纵轴值），不传时记录只记录次数
logbrowser
1/0 -- 是否区分浏览器做记录，默认0
ignoreSpider
1/0 -- 是否忽略爬虫 的请求，1表示不记录爬虫的访问，默认0
count int -- 默认是1，表示该次请求记录N次count

返回：

接口返回一个1*1像素的gif，大约43字节。

能够区分出的浏览器：

代码 含义
IE6 MSIE 6
IE7 MSIE 7
IE8 MSIE 8
IE9 MSIE 9
IE10 MSIE 10
IE11 MSIE 11
Chrome Chrome
Waterfox Waterfox
Firefox Firefox
Safari Safari
Android Android
iOS iOS
Winphone Windows Phone
UnknownTouch 其他移动设备
Unknown 未知浏览器

例子：

http://m.ued.qunar.com/monitor/log?code=hotel_detail_bktool_render&time=1024

var img = new Image();
    img.onload=function(){
    //防止IE7 GC时在请求未发出去之前abort掉该请求。
    img.onload=null;
    img=null;
};
img.src="http://m.ued.qunar.com/monitor/log?code=hotel_detail_bktool_render&time=1024";
